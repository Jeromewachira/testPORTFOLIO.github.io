- **Max Size**: 5MB per image

- **Formats**: JPEG, PNG, WebP, AVIF
- **Features**: Advanced options, real-time preview
- **Offline**: Works offline as PWA

**3. ImageOptim (macOS)**
- **URL**: [imageoptim.com](https://imageoptim.com)
- **Platform**: macOS desktop app
- **Features**: Drag-and-drop, lossless compression

### Command Line Tools

**1. ImageMagick**
```bash
# Install
brew install imagemagick  # macOS
sudo apt install imagemagick  # Ubuntu

# Resize and compress
magick convert input.jpg -resize 1920x1080 -quality 85 output.jpg
```

**2. cwebp (WebP Converter)**
```bash
# Install
brew install webp  # macOS
sudo apt install webp  # Ubuntu

# Convert to WebP
cwebp -q 85 input.jpg -o output.webp
```

### Writing Best Practices

1. **Keep it concise**: 2-3 sentences per paragraph
2. **Use active voice**: "I developed" vs "was developed"
3. **Include keywords**: Cybersecurity terms for SEO
4. **Show progression**: Highlight learning journey
5. **Be specific**: Mention tools, technologies, results

## üí° Pro Tips

### Design Tips
1. **Consistency**: Use the established color scheme
2. **White Space**: Don't overcrowd sections
3. **Hierarchy**: Clear visual hierarchy with typography
4. **Loading**: Optimize for fast loading times
5. **Mobile First**: Design for mobile, enhance for desktop

### Content Tips
1. **Keywords**: Include cybersecurity keywords naturally
2. **Storytelling**: Tell your transition story clearly
3. **Evidence**: Show, don't just tell (projects, skills)
4. **Updates**: Keep content current and fresh
5. **Contact**: Make it easy to reach you

### Technical Tips
1. **Backup**: Keep backups of all original files
2. **Version Control**: Use Git for all changes
3. **Testing**: Test on real devices, not just DevTools
4. **Performance**: Monitor load times regularly
5. **Security**: Keep dependencies updated
